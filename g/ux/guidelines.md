# UX/UI Guidelines

## Navbar

Don't use radios for toggling display, make the display button a toggle itself:

```
| Show menu |
| Hide menu |
```

do this:

```
(hidden)
| Show:  |
| x Menu |

(shown)
| Show:  |
| v Menu |
```

## Text

Avoid making text bold on mouse over => ugly, distracting
