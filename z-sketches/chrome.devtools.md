# Chrome DevTools

## Shortcuts

- open/close DevTools: `F12`
- open file: `ctrl + p`
- view commands: `ctrl + shift + p` (`?` for help)
- go to line of code: `ctrl + g`
- go to symbol: `ctrl + o`

### Commands

Press `ctrl + shift + p` to display DevTools commands.

Sort JSON: `JSON: Sort Document`
