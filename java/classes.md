# Java - Classes

* A `.java` file has only one public class
* A `.java` filename must match its public class 
* A `.java` file that has a static class can be saved under another name

```txt
Hard.java: class Simple

Hard.java --> compiler --> Simple.class (bytecode)

javac Hard.java
java Simple
```

* `A.java` file with 3 static classes A, B, C --> A.class, B.class, C.class
